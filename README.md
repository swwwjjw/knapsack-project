# knapsack-project
Проект с летней практики СПбПУ, 2026.

Запуск приложение: 
1. Копируем репозиторий: git clone [https://github.com/swwwjjw/knapsack-project/](https://github.com/swwwjjw/knapsack-project.git);
2. Заходим в директорию репозитория: cd knapsack-project; 
3. Загружаем зависимости: pip install -r requirements.txt;
4. Запускаем бенчмарк: python benchmark.py;
5. Заупскаем построение графиков: python visualize.py;
6. Запускаем веб-приложение: uvicorn app:app --reload.
